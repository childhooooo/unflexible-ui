# Unflexible UI

Don't be flexible.

## Usage

### Install

```bash
npm install unflexible-ui
```

### Optional settings

#### Wrap

```css
:root {
  --wrap-width-xl: 1240px;
  --wrap-width-l: 1030px;
  --wrap-width-m: 760px;
  --wrap-width-s: 470px;
  --wrap-width-xs: 94%;
}
```
